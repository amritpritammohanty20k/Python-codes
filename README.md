#  Python Practice Codes

This repository contains **basic Python practice codes** written using **core Python only**.  
It focuses on learning and practicing fundamental concepts like loops, arrays (lists), strings, and conditional statements.


##  Topics Covered

- Loops (`for`, `while`)
- Conditional Statements (`if`, `else`, `elif`)
- Lists (Arrays)
- Strings
- Functions
- Recursion 
- Basic input/output
- Logical and pattern problems
